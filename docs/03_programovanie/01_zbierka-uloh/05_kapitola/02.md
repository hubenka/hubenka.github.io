---
title: 2. Cenzúra
publish: true
---

Prišla tvrdá cenzúra s nariadením, že nikto už nesmie vidieť žiadnu samohlásku. Nahraď každý priestupok vo vstupnom texte iným špeciálnym znakom.

**Vstup:**
```
Správa: Ja som tvoj kamarat
Samohlásku nahraď: *
```

**Výstup:**
```
Cenzurované: J* s*m tv*j k*m*r*t
```
